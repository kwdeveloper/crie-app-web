Crie aplicativos web com HTML, CSS, JavaScript, PHP, PostgreSQL, Bootstrap, AngularJS e Laravel
***********************************************************************************************

Relação dos projetos por capítulo
---------------------------------
Pastas dos projetos na ordem em que são apresentados no livro.

Capítulo 2 - Ferramentas de desenvolvimento
  html-format-2

Capítulo 3 - HTML
  html-basics: elementos HTML (lista simples, lista enumerada e tabela).
  html-format: formatação CSS para elementos HTML.
  html-input: entrada de dados (text e password) e submissão de dados (submit e reset) com formulário HTML.
  html-input-2: entrada de dados combo-box, seletor de rádio, seleção múltipla e caixa de texto.

Capítulo 4 - CSS
  html-format-2: formatação CSS para criar tabela "zebrada".
  html-format-regex: formatação CSS baseada em expressões regulares.

Capítulo 5 - Bootstrap
  bootstrap-grid: divisão de uma página HTML em colunas.
  bootstrap-grid-verne: divisão de uma página HTML em colunas.
  bootstrap-quote: formatação do marcador blockquote.
  bootstrap-btn: formatação de botões.
  bootstrap-table: formatação de tabela.
  bootstrap-menu-vert: menu vertical.
  bootstrap-menu-hor: menu horizontal.
  bootstrap-carousel: apresentação de slides (carrossel).

Capítulo 6 - JavaScript
  javascript-hello: exemplo básico "hello, world".
  javascript-decimal: biblioteca para representação decimal de números.
  javascript-escopo: escopo de variável.
  javascript-namespace: uso e criação de ambientes de nomes.
  javascript-ifelse: estrutura de decisão.
  javascript-ifelseif: estrutura de decisão.
  javascript-switch-case: estrutura de decisão.
  javascript-switch-sembreak: estrutura de decisão.
  javascript-for: estrutura de repetição.
  javascript-heranca: implementação de herança com JavaScript.
  javascript-mootools: implementação de herança com a biblioteca Mootools.

Capítulo 7 - jQuery
  jquery-event-window: ação vinculada ao redimensionamento da janela do navegador.
  jquery-event-kbd: ação vinculada ao acionamento do teclado.
  jquery-events-3: ação vinculada à passagem do mouse sobre o elemento.
  jquery-sort: ordenação de uma lista.
  jquery-children: seleção dos filhos de um elemento.
  jquery-toggle: controle de visibilidade de um elemento.
  jquery-font-size: controle da formatação de um elemento.

Capítulo 8 - Gráficos com JavaScript
  jqplot-1: gráfico de linhas usando a biblioteca jqPlot.
  jqplot-2: gráfico de linhas com legendas usando a biblioteca jqPlot.
  Chart.js-pie: gráficos de setores usando a biblioteca Chart.js.
  Chart.js-radar: gráfico do tipo radar usando a biblioteca Chart.js.
  C3-1: gráfico de linhas usando a biblioteca vetorial C3.
  C3-2: gráfico de linhas com legendas usando a biblioteca vetorial C3.

Capítulo 9 - PHP
  php-hello: exemplo básico "hello, world".
  php-tipos: tipos de dados do PHP.
  php-array-1: inclusão de elemento em array.
  php-array-2: inclusão de elemento em array.
  php-array-3: inclusão de elemento em array.
  php-array-sort-1: ordenação de array com asort().
  php-array-sort-2: ordenação de array com sort().
  php-array-sort-3: ordenação de array com uasort().
  php-while: estrutura de repetição.
  php-loop-for: estrutura de repetição.
  php-get: recuperação de dados do cliente usando a variável $_GET.
  php-classes: programação OO com PHP.
  php-classes-heranca: programação OO com PHP (herança).
  php-classes-heranca-2: programação OO com PHP (classe abstrata).
  php-autoload: uso de __autoload() para carregar uma classe.
  php-namespace: definição de um ambiente de nomes.
  php-namespace-2: definição de um ambiente de nomes.

Capítulo 10 - PHP - Acesso a banco de dados
  php-db-sqlite: operações básicas com o SQLite.

Capítulo 11 - Aplicativo Ajax com PHP e JavaScript
  php-cadastrar-cidades:  preenchimento de uma tabela de banco de dados a partir de um arquivo CSV contendo informações sobre as cidades do Brasil.
  php-xml-parser: preenchimento de uma tabela de banco de dados a partir de um arquivo XML contendo informações sobre as cidades do Brasil.
  php-ajax-jquery: consulta à tabela de cidades usando a técnica Ajax.

Capítulo 12 - Laravel
  pessoas: aplicativo de cadastro com informações sobre pessoas.

Capítulo 13 - Laravel – Desenvolvimento de um aplicativo
  cidades: aplicativo com informações sobre as cidades do Brasil incluindo a localização geográfica via Google Maps.

Capítulo 14 - AngularJS
  angular-01: exemplo básico.
  angular-ngbind: uso da diretiva ng-bind.
  angular-02: soma de dois valores.
  angular-notes: aplicativo gerenciador de anotações.
  angular-filter-inverter: filtragem de dados.
  angular-filter-inverter-2: filtragem de dados.
  angular-pagination-numeros: paginação de dados.
  angular-pagination-watch: paginação de dados.
  angular-notes-bd: aplicativo gerenciador de anotações com gravação em banco de dados.

Capítulo 15 - Autenticação e serviço de nuvem
  angular-notes-auth: aplicativo gerenciador de anotações com gravação em banco de dados e autenticação via API do Facebook.
